# NewsEvent Manual Drop-zone — P29B Scaffold

**Status:** OPERATOR_PENDING — awaiting real source files
**PIT Gate:** `publishedAt` (NOT `ingestedAt`)
**entersAlphaScore:** false — this zone is intake-only, no scoring until P29-F dry-run gate passes

---

## Purpose

This directory is the manual drop-zone for Taiwan-listed company news event data
collected by the operator from authorised public sources (e.g., TWSE announcements,
verified financial news publishers).

Files placed here are **NOT automatically imported**. They must pass the P29-F Dry-run Gate
before any corpus or scoring pipeline receives them.

---

## PIT Safety Rule

**Only `publishedAt` may be used as the PIT gate.**

`ingestedAt` (the timestamp when a system scraped or received the news) is NOT the
original publication time and must never be used as the PIT anchor. Using `ingestedAt`
risks treating events as known before they were publicly available.

---

## What to Place Here

- One CSV or JSON file per source batch
- Each file must conform to `EXPECTED_SCHEMA.json`
- Each file must be listed in a `SOURCE_MANIFEST.json` (use `SOURCE_MANIFEST_TEMPLATE.json` as starting point)
- All events must be real, verifiable, publicly published news — no synthetic/mock events

## What NOT to Place Here

- Any file with `DO_NOT_IMPORT` in the filename (those are templates)
- The `SOURCE_MANIFEST_TEMPLATE.json` file itself
- Files containing `approvalToken` (token must be generated by the import gate, not pre-filled)
- Mock events, synthetic events, or events from simulation sources
- Events with pre-assigned sentiment labels or future price reaction labels
- Events without a verifiable `sourceUrl`

---

## Import Flow (do NOT skip steps)

```
1. Operator places file(s) here
2. Operator fills SOURCE_MANIFEST.json (copy from template, remove _TEMPLATE suffix)
3. Operator runs operator QA checklist (docs/manual-data/news-event/P29D_OPERATOR_FILE_QA_CHECKLIST.md)
4. P29-F Dry-run Gate validates schema, PIT gate, deduplication, no-mock check
5. If gate passes → import proceeds under separate approval
6. entersAlphaScore remains false until explicit P29-G gate
```
